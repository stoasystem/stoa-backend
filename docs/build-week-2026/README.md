# OpenAI Build Week 2026 Submission Pack

This directory contains the working materials for preparing STOA's OpenAI Build Week
submission. It is an internal preparation pack, not proof that registration or submission has
been completed.

## Recommended submission identity

- **Project name:** STOA Safe Learning Layer
- **Track:** Education
- **Tagline:** Safe AI tutoring, with humans in the loop.
- **Submission scope:** The security, privacy, practice-integrity, and deterministic delivery
  work added after the Build Week submission period opened—not the whole pre-existing STOA
  platform.

## Files

- [DEVPOST-SUBMISSION.md](DEVPOST-SUBMISSION.md) — paste-ready English submission copy.
- [BUILD-WEEK-EVIDENCE.md](BUILD-WEEK-EVIDENCE.md) — pre-existing/new-work boundary and
  technical evidence.
- [DEMO-SCRIPT.md](DEMO-SCRIPT.md) — storyboard and narration for the required public video.
- [JUDGING-ACCESS.md](JUDGING-ACCESS.md) — repository, demo, test-account, and privacy plan.
- [SUBMISSION-CHECKLIST.md](SUBMISSION-CHECKLIST.md) — blockers, owner-supplied fields, and
  final compliance checks.
- [OWNER-HANDOFF.zh-CN.md](OWNER-HANDOFF.zh-CN.md) — 需要项目负责人确认或提供的中文清单。

## Current decision

The project is eligible in principle as a meaningfully extended pre-existing project. It is not
submission-ready until every item marked `BLOCKER` in `SUBMISSION-CHECKLIST.md` has been closed.

The deadline is **July 21, 2026 at 5:00 PM Pacific Time**, equivalent to **July 22, 2026 at
02:00 Europe/Zurich**.
