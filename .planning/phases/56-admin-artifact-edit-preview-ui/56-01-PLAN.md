# Plan 56-01: Admin Artifact Edit Preview UI

## Objective

Add selected-report artifact edit preview and apply controls to the admin report operations UI while preserving the Phase 54/55 privacy and mutation boundaries.

## Tasks

1. Add typed admin API client support for artifact edit preview/apply routes.
2. Add React Query mutations for preview and apply operations.
3. Add selected-report UI controls for reason, summary, recommendations, preview, sanitized diff, apply, and operation outcome.
4. Keep preview and apply visually distinct, with apply dependent on a successful preview.
5. Extend Playwright coverage for preview/apply flow and private marker denylist.
