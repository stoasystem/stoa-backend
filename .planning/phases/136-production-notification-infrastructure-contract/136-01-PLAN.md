Create a concrete production notification delivery contract that Phase 137 and Phase 138 can implement.

## Work Items

4. Define delivery state fields and operator-visible health/status needs.
5. Record backend, CDK, frontend, and native ownership boundaries.
6. Update phase verification with the contract result and implementation handoff.
