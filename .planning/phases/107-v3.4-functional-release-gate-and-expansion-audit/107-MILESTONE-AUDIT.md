# v3.4 Milestone Audit

**Milestone:** v3.4 Learning Expansion Foundation
**Status:** Passed
**Completed:** 2026-06-08

## Goal

Prepare Phase 2 learning expansion without jumping directly into broad curriculum rollout.

## Requirements

| Requirement | Status | Evidence |
|-------------|--------|----------|
| LEARN-01 Multi-Subject Taxonomy And Prompt Contract | Complete | Phase 104 contract and verification |
| LEARN-02 Backend Subject/Topic Support And Student Profile Seeds | Complete | Phase 105 backend services/routes/tests |
| UI-19 Student And Parent Learning Profile UI | Complete | Phase 106 frontend UI and E2E |
| VERIFY-17 v3.4 Functional Release Gate And Expansion Audit | Complete | Phase 107 release gate |

## Delivered

- Bounded subject taxonomy for math, physics, German, and English.
- Rollout-state language that keeps math active and other subjects in foundation scope.
- Subject-specific prompt context for AI answers.
- Normalized subject validation and topic seed storage on questions.
- Student and parent learning profile endpoints with subject activity and weak-topic evidence.
- Student/parent profile UI for subject signals and freshness.
- Chat/question start UI for rollout-aware subject choice.
- Focused backend and frontend automated verification.

## Deferred Phase 2 Expansions

- Stripe/TWINT payment-provider integration.
- Full multi-subject curriculum content and exercises.
- Student memory/personalization beyond profile seeds.
- AI teacher assistance tools such as summaries and exercise generation.
- WebSocket realtime notifications.
- Mobile responsive polish.
- Full frontend multilingual rollout.
- Support-ticket/evidence integrations after approved connector or credential paths exist.

## Conclusion

v3.4 satisfies the learning expansion foundation locally. The product now has a stable subject/topic/profile base for later curriculum, personalization, teacher assistance, and realtime milestones without claiming those broader capabilities are complete.
