# Plan 68-01: Admin Support Handoff UI

## Objective

Add admin UI controls for support-safe handoff packages backed by the Phase 67 API, with preview/copy/download/refusal states and privacy-safe rendering.

## Tasks

1. Add TypeScript types and API client function for `POST /admin/reports/support-handoff-package`.
2. Add React Query mutation hook.
3. Extend `ReportOperationsPage` state and handlers for handoff reason, note, destination mode, evidence inclusion toggles, result message, copy, and download.
4. Add `SupportHandoffPanel` using existing card/control styling.
5. Update admin report operations E2E route mocks and assertions for preview/copy/download/refusal and privacy denylist.
6. Run frontend lint/build and focused Playwright where feasible.
7. Update GSD verification/summary and milestone progress.

## Verification

- `npm run lint`.
- `npm run build`.
