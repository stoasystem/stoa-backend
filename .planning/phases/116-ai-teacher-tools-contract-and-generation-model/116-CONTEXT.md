# Phase 116 Context: AI Teacher Tools Contract And Generation Model

**Milestone:** v3.7 AI Teacher Tools And Exercise Generation
**Requirement:** AITOOL-01
**Status:** Planned

## Why This Phase Exists

`stoa_docs` Phase 2 calls for AI teacher assistance tools such as automatic summaries and exercise generation. Earlier milestones created subject/topic taxonomy, learning profile seeds, teacher summary seeds, notification events, and realtime delivery. Phase 116 defines the generation contract before backend/UI implementation.

## Product Scope

- Automatic session and misconception summaries.
- Suggested teaching focus and draft follow-up explanation.
- Bounded exercise draft generation by subject, topic, difficulty, and count.
- Review lifecycle for accept/reject/archive/regenerate.
- Explicit no-auto-send boundary.

## Completion Criteria

Phase 116 completes when the AI tool contract, draft lifecycle, generation inputs/outputs, UI implications, and functional verification checklist are written and internally consistent.
