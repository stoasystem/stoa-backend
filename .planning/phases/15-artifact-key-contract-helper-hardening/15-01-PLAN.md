# Plan 15-01: Artifact Helper Contract Hardening

**Phase:** 15 - Artifact Key Contract & Helper Hardening
**Status:** Planned
**Requirements:** ARTIFACT-01, ARTIFACT-02, ARTIFACT-03, ARTIFACT-04, ARTIFACT-05, STORAGE-01, STORAGE-02, STORAGE-03, STORAGE-04, STORAGE-08

## Objective

Replace embedded report artifact key/write behavior with a small helper module that enforces the canonical private S3 artifact contract and supports future smoke reads.

## Tasks

1. Add `src/stoa/services/report_artifact_service.py`.
   - Define canonical prefix and content type constants.
   - Expose `build_report_artifact_keys(parent_id, student_id, week_start)`.
   - Expose `write_report_artifacts(json_artifact, html_artifact, keys, s3_client=None)`.
   - Expose `get_report_json(s3_key, s3_client=None)`.

2. Harden key validation.
   - Reject blank parent IDs, student IDs, and week starts.
   - Reject segments containing slashes, spaces, `@`, or other non-backend-ID characters.
   - Validate and normalize `week_start` with ISO date parsing.
   - Keep canonical IDs in the key; do not sanitize emails or display names into plausible paths.

3. Wire weekly report storage through the helper.
   - Replace `_report_artifact_keys` usage with helper calls.
   - Replace inline `put_object` calls with helper writes.
   - Preserve JSON-before-HTML write order and metadata-after-write behavior.

4. Add focused tests.
   - Assert exact JSON and HTML keys.
   - Assert invalid/blank/email/display-name/slash inputs fail closed.
   - Assert S3 writes use the configured reports bucket, correct content types, and no ACL parameter.
   - Assert JSON read helper decodes S3 body content by key.
   - Keep report flow tests passing.

## Verification

## Acceptance Criteria

- ARTIFACT-01/02: exact key shape is helper-backed and tested.
- ARTIFACT-03/04/05: keys use safe backend IDs + ISO week start and reject unsafe values.
- STORAGE-01/02/03/04: helper exposes testable writes with required content types and no ACL.
- STORAGE-08: helper can read and decode a JSON artifact by S3 key.
