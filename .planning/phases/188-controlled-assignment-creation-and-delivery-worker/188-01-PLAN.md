# Plan 188-01: Implement Controlled Assignment Creation And Delivery Worker

## Goal

Create reviewed assignments from approved automation batches without duplicate side effects.

## Work Items

1. Add a tutor/admin-only batch execution request and route.
2. Normalize submitted policy and selected candidates using the Phase 187 contract.
3. Rehydrate assignment sources through the existing reviewed assignment source path.
4. Add automation metadata to created assignments: policy ID, batch ID, candidate ID, automation level, actor, source attribution, and result evidence.
5. Make execution idempotent by deterministic automation key and return duplicate/skipped/refused/failed item results.
6. Preserve role-safe assignment responses for tutor/admin, student, and parent views.
7. Add focused tests for idempotency, partial batch behavior, source attribution, duplicate prevention, and visibility.
