# Plan 197-01: Implement Dispatch Planner And Candidate Ranking

## Goal

Add a non-mutating planner that ranks eligible teachers/tutors for escalated questions and returns selected/refused candidates with stable reason codes.

## Work Items

1. Add a dispatch service with teacher profile normalization.
2. Add planner refusal rules for role eligibility, availability, max load, subject capability, and previous timeout.
3. Add deterministic ranking by load, recent SLA health, last dispatch fairness, and teacher ID.
4. Add stable planner response shape for operator preview and tests.
5. Add focused tests for selected/refused candidates and no-candidate reasons.
