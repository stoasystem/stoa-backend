# Phase 52 Plan

## Scope

Add report editing controls to `/admin/report-operations`:

- API client functions and types for edit draft create/read/apply.
- React Query mutation hooks for create/apply.
- Selected report detail UI with reason, editable metadata fields, create draft, apply draft, and result state.
- Playwright e2e coverage for the create/apply workflow and privacy denylist.

## Safety Gates

- UI must not render edit controls without a selected report.
- Create draft and apply draft must be visibly separate controls.
- UI and tests must keep denylist clean for `weekly-reports/`, S3 key fields, presigned URLs, and S3 URLs.
