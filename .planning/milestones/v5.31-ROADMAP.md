**Status:** Contract Complete

## Goal

Execute the approved limited pilot cohort with live operational monitoring, support coverage, feedback capture, and daily go/hold/rollback decisions.

## Function Purpose

- Validate STOA with real parent/student users under controlled scope.
- Measure actual activation, first learning action, support load, AI quality, mobile stability, billing/account friction, and satisfaction.
- Produce real outcome evidence before any expansion.

## Implementation Strategy

- Start only if v5.30 returns `start_limited_pilot`.
- Keep cohort narrow, flags staged, and rollback immediate.
- Operate daily launch-room review with support and incident owners.
- Close with continue/pause/rollback/remediate decision.
