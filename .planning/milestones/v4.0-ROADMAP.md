v4.0 is complete when:

- Adaptive memory and reviewed assignment contracts are documented.
- Backend memory/recommendation/assignment APIs work with curriculum and AI draft sources.
- Student/tutor/parent UI uses real backend data for recommendations, assignments, and progress signals.
- Focused backend/frontend checks and browser smoke prove the assignment workflow works.
- `STOA_DOCS_FEATURE_GAP_AUDIT.md` records the personalization outcome and residual autonomous tutoring scope.
