## Release Gate

v3.9 is complete when:

- Payment provider contract, subscription state model, and webhook lifecycle are documented.
- Backend checkout/status/webhook APIs work in sandbox/test mode and preserve manual override compatibility.
- Parent/admin UI uses real backend billing APIs.
- Focused backend/frontend checks and browser smoke prove the payment workflow surfaces work.
- `STOA_DOCS_FEATURE_GAP_AUDIT.md` records the payment integration outcome and residual live-charge/tax/accounting scope.
