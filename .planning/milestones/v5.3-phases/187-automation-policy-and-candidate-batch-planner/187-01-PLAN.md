# Plan 187-01: Implement Automation Policy And Candidate Batch Planner

## Goal

Convert v5.2 recommendations into policy-bounded assignment candidate batches without creating student work.

## Work Items

2. Normalize policy fields for source type, subject/topic filters, confidence, freshness, max assignment count, delivery mode, and paused/off states.
3. Reuse existing adaptive recommendations and assignment signal state for candidate ranking and duplicate refusal.
4. Add a route contract for previewing assignment automation batches.
5. Return stable selected/refused/summary response fields with review boundaries preserved.
6. Add focused tests for policy source filtering, paused-policy refusal, no assignment side effects, and response shape.
