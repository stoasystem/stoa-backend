# Roadmap: v5.26 Limited Pilot Execution And Outcome Evidence

**Status:** Contract Complete
**Created:** 2026-07-07
**Prior milestone:** v5.25 Pilot Activation Blocker Burn-Down And Safe Start Decision

## Goal

Run the approved limited pilot cohort under controlled rollout, daily monitoring, support coverage, feedback capture, and evidence-based continuation/rollback decisions.

## Function Purpose

- Move from launch readiness contracts into real pilot operation.
- Measure whether parents/students can activate, use, get help, and keep using STOA under controlled conditions.
- Generate outcome evidence that can justify continuation, remediation, rollback, or expansion.

## Implementation Strategy

- Start only if v5.25 safe-start gate says `start`.
- Keep cohort narrow and feature flags staged.
- Use daily operational review instead of waiting until the end of the pilot.
- Treat support load, account friction, AI quality, mobile stability, and billing/entitlement problems as product signals, not anecdotes.
- Close with an evidence-backed continue/pause/rollback/remediate decision.
