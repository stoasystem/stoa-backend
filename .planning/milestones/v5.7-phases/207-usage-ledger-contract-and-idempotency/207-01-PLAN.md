<objective>
Create the durable usage ledger contract and idempotency primitives for question usage.
</objective>

<tasks>
1. Add a usage ledger repository with conditional event writes, deterministic key lookup, period listing, and counter reads.
2. Add service helpers for question usage event construction, idempotency key resolution, privacy-safe entitlement snapshots, and UTC quota period handling.
3. Extend question submission request parsing with optional `idempotencyKey`.
4. Add tests proving duplicate ledger writes are idempotent and privacy-sensitive fields are excluded.
</tasks>
