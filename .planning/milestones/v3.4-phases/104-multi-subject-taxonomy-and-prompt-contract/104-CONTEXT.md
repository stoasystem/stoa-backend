# Phase 104 Context: Multi-Subject Taxonomy And Prompt Contract

**Milestone:** v3.4 Learning Expansion Foundation
**Requirement:** LEARN-01
**Status:** Planned

## Why This Phase Exists

`stoa_docs` Phase 2 calls for multi-subject support, student memory/personalization, and AI teacher assistance. The current MVP is math-first. Phase 104 defines the stable taxonomy, prompt behavior, and rollout boundaries before backend/UI implementation.

## Product Scope

- Subject identifiers and rollout state for math, physics, German, and English.
- Topic/knowledge-point shape for summaries, weak-topic tracking, and future exercises.
- Subject-specific AI prompt expectations.
- Backward compatibility for existing math questions and reports.

## Completion Criteria

Phase 104 completes when the taxonomy contract, prompt contract, data model assumptions, UI implications, and lightweight verification checklist are written and internally consistent.
