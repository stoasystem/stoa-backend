# Phase 120 Context: Full Curriculum Rollout Contract And Content Model

**Milestone:** v3.8 Full Curriculum Rollout
**Requirement:** CURRIC-01
**Status:** Planned

## Why This Phase Exists

`stoa_docs` Phase 2 calls for broader curriculum coverage beyond the initial math-focused practice flows. Earlier milestones created subject/topic foundations, learning profile seeds, practice progress, and reviewed exercise draft generation. Phase 120 defines the curriculum content model and rollout contract before backend/UI implementation.

## Product Scope

- Curriculum hierarchy across subject, grade/band, unit, topic, lesson, exercise, and checkpoint.
- Full rollout target subjects: math, physics, German, and English.
- Exercise bank metadata and answer/explanation boundaries.
- Content lifecycle states: seed, draft, reviewed, active, archived.
- Compatibility with existing practice progress, mistakes, lessons, challenges, and attempts.

## Completion Criteria

Phase 120 completes when the curriculum hierarchy, content fields, rollout states, migration/backfill expectations, UI implications, and functional verification checklist are written and internally consistent.
