# Phase 96 Context: Content Moderation Contract And Data Model

**Milestone:** v3.2 Content Moderation And Internal Operations
**Requirement:** MOD-01
**Status:** Planned

## Why This Phase Exists

After v3.1, the only remaining visible MVP admin workflow in `stoa_docs` is content moderation: admins need a way to review reported or abnormal learning content. Phase 96 defines the minimal product contract before backend and UI implementation.

## Product Scope

- Reportable surfaces: question content, AI answer, teacher reply, and optional reporter note.
- Admin workflow: queue, filters, detail, assignment, status transition, resolution note.
- Internal development mode: prioritize usable workflow over broad compliance evidence.

## Completion Criteria

Phase 96 completes when the moderation case contract, data model, API plan, UI workflow, and lightweight verification checklist are written and internally consistent.
