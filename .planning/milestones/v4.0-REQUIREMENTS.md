# Requirements: v4.0 Adaptive Learning Memory And Assignment

**Milestone:** v4.0
**Status:** Active
**Created:** 2026-06-10

## Goal

Build durable learning memory, next-practice recommendations, and reviewed assignment workflows on top of curriculum progress, questions, feedback, tutor replies, and AI exercise drafts.

## Requirements

| Requirement | Phase | Status |
|-------------|-------|--------|
| ADAPT-01 Adaptive learning memory and assignment contract | Phase 128 | Planned |
| ADAPT-02 Backend learning memory and reviewed assignment APIs | Phase 129 | Planned |

| VERIFY-23 v4.0 functional release gate and personalization audit | Phase 131 | Planned |

## Product Build Priority

This milestone closes the most valuable remaining `stoa_docs` Phase 2 learning gap: personalization beyond profile seeds. It should produce visible student/tutor/parent workflows, not only background analytics.

## Out Of Scope

- Fully autonomous tutoring decisions.
- Automatically assigning generated exercises without review.
- Native mobile apps.
- Broad compliance/security program beyond normal authorization and functional correctness.
